如何在Lean4中写一个XOR类型

发表于 2024-02-23

逻辑连词XOR(异或)(exclusive or)，写作

A⊕BA\oplus B A⊕B

真值表是

AAA BBB A⊕BA\oplus BA⊕B
⊤\top⊤ ⊤\top⊤ ⊥\bot⊥
⊤\top⊤ ⊥\bot⊥ ⊤\top⊤
⊥\bot⊥ ⊤\top⊤ ⊤\top⊤
⊥\bot⊥ ⊥\bot⊥ ⊥\bot⊥

XOR的性质：

  • 输入不同则为真，相同则为假
  • A⊕BA\oplus BA⊕B为真则意味着AAA和BBB刚好一真一假
  • 交换律：A⊕B=B⊕AA\oplus B = B\oplus AA⊕B=B⊕A
  • 结合律：(A⊕B)⊕C=A⊕(B⊕C)(A\oplus B)\oplus C = A\oplus (B\oplus C)(A⊕B)⊕C=A⊕(B⊕C)
